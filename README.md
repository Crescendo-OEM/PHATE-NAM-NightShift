# Deep Learning-Based Physiological Risk Stratification in Night-Shift Hospital Workers

This repository contains the official Python implementation of the analytical pipeline for the manuscript: "Deep Learning-Based Physiological Risk Stratification in Night-Shift Hospital Workers" (Accepted for publication in Scientific Reports).

## Overview
This study introduces a novel explainable AI framework that combines nonlinear manifold learning with deep learning to stratify and interpret physiological risks among night-shift hospital workers. 

The pipeline consists of four sequential steps:
1. Nonlinear Manifold Embedding (PHATE): Constructs a global physiological topology based on routine biomarkers.
2. Global Risk Mapping (LightGBM): Estimates the metabolic risk probability and visualizes occupational gradients among the night-shift subset.
3. Interpretable Risk Learning (NAM): Employs a Neural Additive Model (PyTorch) to extract non-linear feature contributions and partial dependence plots (PDPs) without sacrificing interpretability.
4. Latent Clustering (VaDE/GMM): Identifies high-risk phenotypes within the latent space and visualizes the relative risk topography.

## Repository Structure

- /data
  - Place your input dataset here as `dataset.xlsx`. (Note: Due to privacy and ethical restrictions, the raw clinical data is not provided in this public repository. Researchers should place their formatted dataset or a mock dataset here to run the pipeline).
  - The script will generate `processed_dataset.csv` in this folder during runtime.
- /results
  - All generated high-resolution figures (Figure 2 to Figure 8) will be automatically saved here.
- 01_Data_Preprocessing_and_PHATE.py
- 02_LightGBM_Risk_Mapping.py
- 03_NAM_Deep_Learning.py
- 04_VaDE_Clustering.py

## Requirements
The code is written in Python 3. Ensure you have the following main libraries installed:

pip install numpy pandas matplotlib seaborn scikit-learn phate lightgbm torch

## How to Run
The scripts must be executed sequentially, as downstream scripts depend on the processed data and probabilities generated by the upstream scripts.

Step 1: Run Preprocessing and PHATE
python 01_Data_Preprocessing_and_PHATE.py

Step 2: Run Risk Probability Mapping
python 02_LightGBM_Risk_Mapping.py

Step 3: Run NAM Deep Learning
python 03_NAM_Deep_Learning.py

Step 4: Run Latent Clustering
python 04_VaDE_Clustering.py
